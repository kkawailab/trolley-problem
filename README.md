# トロッコ問題 - 倫理的ジレンマゲーム

古典的な哲学的思考実験「トロッコ問題」をインタラクティブなゲーム体験として再現したWebアプリケーションです。プレイヤーは困難な倫理的選択を迫られ、その決定を通じて功利主義や義務論などの倫理学的概念について考察します。

## 概要

このアプリケーションは、避けられない状況下での道徳的判断をシミュレートします。プレイヤーは車を運転している状況で、左右どちらかに曲がることで異なる結果をもたらす選択を迫られます。全ての選択肢には犠牲が伴い、プレイヤーは自身の価値観と向き合うことになります。

## 主な特徴

- **多様なシナリオ**: 5つの異なる倫理的ジレンマを収録
- **リアルタイム計測**: 決定までの時間を記録し、判断プロセスを可視化
- **統計分析**: 他のプレイヤーの選択と比較し、集団的傾向を確認
- **教育的コンテンツ**: 各選択後に倫理的考察ポイントを提供
- **日本語完全対応**: すべてのUIとコンテンツが日本語

## 収録シナリオ

1. **古典的なトロッコ問題** - 5人を救うために1人を犠牲にするか
2. **家族か他人か** - 10人の他人と最愛の家族、どちらを選ぶか
3. **若者か高齢者か** - 命の価値は年齢で変わるのか
4. **法を守る人か破る人か** - ルール違反者と遵守者、誰を救うか
5. **医師か一般人か** - 社会的価値を判断基準にすべきか

## 使い方

1. Webブラウザで `index.html` を開く
2. 「ゲームを開始」ボタンをクリック
3. 各シナリオで左右いずれかの選択肢を選ぶ
   - マウスクリックまたは矢印キー（←→）で選択可能
4. 結果と統計を確認
5. 全5シナリオ完了後、総合結果を表示

## 技術仕様

- **HTML5**: セマンティックなマークアップ
- **CSS3**: レスポンシブデザイン、アニメーション
